# Docs Keyboard & Mouse Actions

Implement word-processor-level keyboard and mouse actions for the Canvas-based document editor.

## High Priority

- [x] 1. Word boundary utility (find next/previous word boundary in text)
- [x] 2. Ctrl/Option+Arrow word-by-word cursor movement
- [x] 3. Double-click to select word
- [x] 4. Triple-click to select paragraph
- [x] 5. Shift+Click to extend selection
- [x] 6. Ctrl/Option+Backspace/Delete to delete word
- [x] 7. Clipboard support (copy / paste / cut)

## Medium Priority

- [x] 8. Cmd/Ctrl+Home/End to jump to document start/end
- [x] 9. Cmd+Left/Right to jump to line start/end (Mac)
- [x] 10. Select to document start/end (Cmd/Ctrl+Shift+Home/End)
- [x] 11. Cmd+Backspace to delete to line start
- [ ] 12. Alt+Up/Down to jump to paragraph start/end
- [ ] 13. Page Up / Page Down
- [ ] 14. Shift+Enter for soft line break

## Lower Priority

- [x] 15. Strikethrough toggle (Cmd/Ctrl+Shift+X)
- [x] 16. Clear formatting (Cmd/Ctrl+\)
- [ ] 17. Ctrl+K to delete to line end (Mac)

## Future

- [ ] 18. Undo/redo selection restore (save cursor + selection in snapshot)
