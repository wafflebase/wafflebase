# CLI Package ‚Äî Task

## Goal
Create `@wafflebase/cli` TypeScript package in the pnpm monorepo with agent-friendly
features. Design doc: `docs/design/rest-api-and-cli.md` sections 7‚Äì8.

Depends on: Backend REST API task (20260312-backend-rest-api-todo.md).

## Phase 1: Package Scaffold

- [ ] Create `packages/cli/` directory
- [ ] Create `package.json` ‚Äî name: `@wafflebase/cli`, bin: `wafflebase`, deps: `@wafflebase/sheet`, `commander`, `yaml`
- [ ] Create `tsconfig.json` ‚Äî ESM, target ES2022, moduleResolution bundler
- [ ] Create `vitest.config.ts`
- [ ] Create `eslint.config.js` (match frontend pattern)
- [ ] Create `src/bin.ts` ‚Äî entry point with `#!/usr/bin/env node`

## Phase 2: Config + HTTP Client + Output

- [ ] Create `src/config/config.ts` ‚Äî load `~/.config/wafflebase/config.yaml`, env vars, flag precedence
- [ ] Create `src/client/http-client.ts` ‚Äî REST API v1 wrapper using built-in fetch
- [ ] Create `src/client/dry-run.ts` ‚Äî print request without executing when `--dry-run` is set
- [ ] Create `src/client/types.ts` ‚Äî re-export relevant types from `@wafflebase/sheet`
- [ ] Create `src/output/formatter.ts` ‚Äî format dispatcher (json/table/csv/yaml)
- [ ] Create `src/output/json.ts` ‚Äî JSON formatter (pretty-print, error format)
- [ ] Create `src/output/table.ts` ‚Äî table formatter (aligned columns)
- [ ] Create `src/output/csv.ts` ‚Äî CSV formatter
- [ ] Create `src/output/yaml.ts` ‚Äî YAML formatter
- [ ] Create `src/commands/root.ts` ‚Äî root program with global flags (--server, --api-key, --workspace, --profile, --format, --quiet, --verbose, --dry-run)
- [ ] Write unit tests for config resolution and output formatters
