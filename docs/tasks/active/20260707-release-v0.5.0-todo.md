# Release v0.5.0

Close out the v0.5.0 release covering the `v0.4.9..HEAD` window (4
commits). Theme: **PDF viewer** ‚Äî a fourth static document type (upload a
PDF, store the original as an S3/MinIO blob, view it read-only in a pdf.js
viewer at `/f/:id`, document-permission-gated serving; the `pdf-` Yorkie
key + comments/presence reserved for Phase 2). Plus documents-list
usability (clickable-header sorting, type-filter chips, Yorkie-derived
last-modified, owner column) and a CLI packaging fix (bundle the private
`@wafflebase/slides`/`tokens` workspace deps into `dist/bin.js` so
`npm install -g @wafflebase/cli` stops 404ing).

Minor (not patch) bump: PDF viewer introduces a **new document type** and
**new infra** ‚Äî a blob store (`FILE_STORAGE_*` env vars, MinIO in dev) and
the `POST /files` + `GET /documents/:id/file` endpoints.
