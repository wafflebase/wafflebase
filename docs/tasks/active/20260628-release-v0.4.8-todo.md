# Release v0.4.8

Close out the v0.4.8 patch release covering the `v0.4.7..HEAD` window
(20 commits). Theme: continued Slides Google-Slides / PowerPoint parity
— OOXML preset shape-geometry fidelity audit, an in-editor theme builder
(customize colors / fonts / background, canvas layout-editing mode, and
editable-share-link toolbar parity), and a Slides CLI with full PPTX
export — plus Docs header/footer tables (import / export / render /
interactive cell editing) with DOCX table follow-ups, curated Google
Fonts embedding, hardened export, and a Sheets datasource result-rendering
improvement.
