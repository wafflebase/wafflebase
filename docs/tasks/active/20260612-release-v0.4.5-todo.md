# Release v0.4.5

## Scope

Cut a **patch** release covering the feature + fix PRs between `v0.4.4`
and `HEAD`. Theme is **Slides editing polish toward Google Slides /
PowerPoint parity**: multi-select resize + unified ghost preview,
group resize that bakes into children (no more glyph distortion),
soft-snap free rotate to 0/90/180/270°, yellow-diamond bend handles
for Elbow Z + Curved connectors, variable pasteboard so off-slide
shapes stay selectable, and shape text-edit font-size fixes. **Slides
Tables** lands as structured editing (P1–P5) + structured PPTX import,
replacing the previous flatten-table import path. **PPTX import**
picks up an ellipse-cardinal connection-site fix and `lumMod` / `lumOff`
color modifier support (slide-21 roadmap fix). **Docs / Slides** share a
Korean font fallback at render, export, and CLI. **Docs** also fixes a
collapsed-caret picker freeze and dropdown focus race. **Sheets +
Docs** open the keyboard-shortcuts help with `Cmd+/`.

### Devops impact

No Prisma schema migration. No backend env vars renamed or added —
this is the routine **image-tag-only** kind of bump per
[`project_wafflebase_release_policy`].
