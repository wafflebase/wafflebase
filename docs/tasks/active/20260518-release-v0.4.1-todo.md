# Release v0.4.1

## Scope

Cut a **patch** release covering the 43 commits between `v0.4.0` and
`HEAD`. Largest theme is **PPTX best-effort import for Slides** plus
follow-up fidelity fixes (shape coverage, image opacity, table cell
margins, blipFill backgrounds, rotated group bbox, connector endpoint
landing). Other notable work: Docs comments (text-range threads),
read-only and Phase-B editable mobile Slides view, Slides presentation
mode v1, Slides connectors, slides toolbar redesign, GA4, several
formula correctness fixes (IPMT/PPMT sign, IRR/MIRR array literals,
OFFSET width/height, TEXT digit placeholders, TRIM whitespace, function
arity validation), and a new wafflebase slides import CLI + backend
writer.

No new server-side infrastructure (no new `process.env.*` reads in
backend, no Prisma schema migration, no new module wiring in
`app.module.ts`/`main.ts`) — this is a **routine devops image bump**.
