# Release v0.6.0

Close out the v0.6.0 release covering the `v0.5.1..HEAD` window (10
commits). Theme: **new Notes document type + shared-core extraction +
Yorkie auth-webhook infra + Slides/Sheets parity**.

**Minor (not patch) bump** — this window introduces a new document type and
new infrastructure, mirroring the v0.5.0 convention (minor reserved for a
new document type + new infra):

- **New document type** — Notes (`"note"`), a collaborative markdown note
  ported from CodePair, with a new `note-<id>` Yorkie docKey (#480).
- **New packages** — `@wafflebase/notes` (the notes engine) and
  `@wafflebase/core` (shared tokens + geometry, absorbs the former
  `@wafflebase/tokens`) (#480, #477).
- **New infra** — per-document access control at the Yorkie layer via a
  backend auth webhook + short-lived minted token (new `YORKIE_*` env
  vars); ships in shadow mode by default (#469).

## Highlights

- **Notes — new document type**: collaborative markdown notes ported from
  CodePair — CodeMirror 6 source editor + live preview, whole content in
  one Yorkie `Text` at `root.content`, new `@wafflebase/notes` engine and
  `note-<id>` docKey; inherits workspaces/sharing/auth-webhook/presence
  (#480).
- **Infra — `@wafflebase/core`**: shared-core extraction PR1 — tokens +
  geometry consolidated into one foundation package (`/tokens`,
  `/geometry` subpath exports), absorbing `@wafflebase/tokens` (#477).
- **Infra — Yorkie auth webhook**: server-enforced per-document read/write
  access at the Yorkie layer; backend-minted short-lived token via
  `authTokenInjector`, HMAC-guarded, shadow→enforce rollout (#469).
- **Slides**: Background panel with Color / Image / Gradient sections
  (#475); font OOXML round-trip fidelity — super/subscript baseline +
  text-run hyperlink export (Phase A + B.1/B.2/B.4) (#474); mobile/presenter
  view repaints on async asset load (#476).
- **Sheets — data validation**: Number & Text criteria (#473); date
  validation with operators + calendar picker (#470); checkbox parity —
  range Space toggle + merged-cell hit-test (#472); checkbox formula-cell
  overwrite + case-sensitivity fix (#471).
