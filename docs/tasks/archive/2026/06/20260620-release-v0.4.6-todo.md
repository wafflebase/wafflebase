# Release v0.4.6

## Scope

Cut a **patch** release covering the feature + fix PRs between `v0.4.5`
and `HEAD`. Theme is again **Slides editing polish toward Google Slides /
PowerPoint parity**. Headline items: a **rich Google Fonts** library
(data-driven catalog, per-font lazy load, "More fonts…" search dialog),
**custom-geometry (freeform) shape** import + render from PPTX, a unified
**object toolbar** with text-box fill/border rendering, **color palette
that stays open** on custom pick plus per-document **recent colors**, and
the **mobile shell** now mounting on the shared view-only route. Editing
robustness improves: text overflow paints while editing, table cell
text-edit boxes stay at full cell size, and `Cmd+Arrow` / clipboard
shortcuts no longer drop text-edit. **Slides collaboration** picks up
live speaker-notes sync on remote Yorkie changes, and **connectors**
remap their endpoints on paste / duplicate. **Slides tables** repair the
merge invariant on cell paste. **Docs** exits the list on Backspace in an
empty list item; **Sheets** scopes its entry loader to the content area
like docs/slides.

### Devops impact

No Prisma schema migration. No backend env vars renamed or added —
this is the routine **image-tag-only** kind of bump per the release
policy.
