# Release v0.4.7

Close out the v0.4.7 patch release covering the `v0.4.6..HEAD` window
(23 commits). Theme: Slides editing polish toward Google Slides /
PowerPoint parity — native Yorkie undo, slide transitions + object
animations, peer-selection presence rings, P0 raster PDF export,
interactive image crop, 23 built-in themes, per-type format effects,
PPTX effects/recolor import, image drag-drop + clipboard paste, faithful
OOXML arrow geometry — plus cross-app `@user` comment mentions and a
frontend a11y/Radix tidy.
