# Release v0.4.9

Close out the v0.4.9 patch release covering the `v0.4.8..HEAD` window
(22 commits). Theme: Docs word-processor polish (spell check + unified
context menu, named styles, missing shortcuts, curated-font DOCX round

cursors, ungroup scale invariant, black-canvas-on-resize fix, PPTX export
in the UI, alpha/none color swatches); Sheets pivot number-format
inheritance plus external-data-source design docs / connector plans.
