Keyboard Shortcuts ​

Keyboard shortcuts for common actions. Mac shortcuts are shown with ⌘, Windows/Linux with Ctrl.

Navigation ​

ActionShortcut
Move to adjacent cellArrow keys
Jump to edge of dataCtrl+Arrow / ⌘+Arrow
Go to cell A1Ctrl+Home / ⌘+Home
Move to next cellTab (right), Enter (down)
Move to previous cellShift+Tab (left), Shift+Enter (up)

Selection ​

ActionShortcut
Extend selectionShift+Arrow
Select to edge of dataCtrl+Shift+Arrow / ⌘+Shift+Arrow
Select entire rowClick row number
Select entire columnClick column header

Editing ​

ActionShortcut
Edit current cellF2 or start typing
Cancel editingEscape
Delete cell contentDelete / Backspace
UndoCtrl+Z / ⌘+Z
RedoCtrl+Y / ⌘+Shift+Z

Clipboard ​

ActionShortcut
CopyCtrl+C / ⌘+C
CutCtrl+X / ⌘+X
PasteCtrl+V / ⌘+V
