# Keyboard Shortcuts

Keyboard shortcuts for the slides editor. Mac shortcuts are shown with `Ōīś`, Windows/Linux with `Ctrl`. Press `Ōīś+/` (or `Ctrl+/`) in the editor to open the in-app shortcuts help.

## Selection

| Action | Shortcut |
|---|---|
| Select all elements on the slide | `Ōīś+A` / `Ctrl+A` |
| Cycle to next element | `Tab` |
| Cycle to previous element | `Shift+Tab` |
| Clear selection / exit text edit | `Esc` |
| Enter text edit on selected text element | `F2` or `Enter` |

## Slides

| Action | Shortcut |
|---|---|
| New slide (after current) | `Ōīś+M` / `Ctrl+M` |
| Duplicate current slide | `Ōīś+Shift+D` / `Ctrl+Shift+D` |
| Next slide | `Page Down` |
| Previous slide | `Page Up` |

## Editing

| Action | Shortcut |
|---|---|
| Undo | `Ōīś+Z` / `Ctrl+Z` |
| Redo | `Ōīś+Shift+Z` / `Ctrl+Shift+Z` |
| Delete selected | `Delete` / `Backspace` |
| Duplicate selected | `Ōīś+D` / `Ctrl+D` |
| Nudge selection | Arrow keys |
| Nudge selection (larger step) | `Shift+Arrow` |

## Clipboard

| Action | Shortcut |
|---|---|
| Copy | `Ōīś+C` / `Ctrl+C` |
| Cut | `Ōīś+X` / `Ctrl+X` |
| Paste | `Ōīś+V` / `Ctrl+V` |
| Paste without formatting | `Ōīś+Shift+V` / `Ctrl+Shift+V` |

## Z-order

| Action | Shortcut |
|---|---|
| Bring to front | `Ōīś+Shift+]` / `Ctrl+Shift+]` |
| Send to back | `Ōīś+Shift+[` / `Ctrl+Shift+[` |
| Bring forward | `Ōīś+]` / `Ctrl+]` |
| Send backward | `Ōīś+[` / `Ctrl+[` |

## Text Editing

These apply while editing inside a text box.

| Action | Shortcut |
|---|---|
| Bold | `Ōīś+B` / `Ctrl+B` |
| Italic | `Ōīś+I` / `Ctrl+I` |
| Underline | `Ōīś+U` / `Ctrl+U` |
| Insert / edit link | `Ōīś+K` / `Ctrl+K` |

## Present

| Action | Shortcut |
|---|---|
| Start from current slide | `Ōīś+Enter` / `Ctrl+Enter` |
| Start from first slide | `Ōīś+Shift+Enter` / `Ctrl+Shift+Enter` |
| Next slide (in present mode) | `ŌåÆ` / `Page Down` / Click |
| Previous slide (in present mode) | `ŌåÉ` / `Page Up` |
| Exit | `Esc` |

## Help

| Action | Shortcut |
|---|---|
| Open shortcuts help | `Ōīś+/` / `Ctrl+/` |
