# Keyboard Shortcuts

Keyboard shortcuts for common actions. Mac shortcuts are shown with `‚åò`, Windows/Linux with `Ctrl`.

## Navigation

| Action | Shortcut |
|--------|----------|
| Move to adjacent cell | Arrow keys |
| Jump to edge of data | `Ctrl+Arrow` / `‚åò+Arrow` |
| Go to cell A1 | `Ctrl+Home` / `‚åò+Home` |
| Move to next cell | `Tab` (right), `Enter` (down) |
| Move to previous cell | `Shift+Tab` (left), `Shift+Enter` (up) |

## Selection

| Action | Shortcut |
|--------|----------|
| Extend selection | `Shift+Arrow` |
| Select to edge of data | `Ctrl+Shift+Arrow` / `‚åò+Shift+Arrow` |
| Select entire row | Click row number |
| Select entire column | Click column header |

## Editing

| Action | Shortcut |
|--------|----------|
| Edit current cell | `F2` or start typing |
| Cancel editing | `Escape` |
| Delete cell content | `Delete` / `Backspace` |
| Undo | `Ctrl+Z` / `‚åò+Z` |
| Redo | `Ctrl+Y` / `‚åò+Shift+Z` |

## Clipboard

| Action | Shortcut |
|--------|----------|
| Copy | `Ctrl+C` / `‚åò+C` |
| Cut | `Ctrl+X` / `‚åò+X` |
| Paste | `Ctrl+V` / `‚åò+V` |
